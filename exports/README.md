# Exports Directory

This directory contains backup/export packages created from the admin panel.

## Security Notes
- Export packages contain your complete database and website files
- Keep exports secure and delete them after transferring to your server
- Do not share exports publicly as they contain sensitive data

## Files
Export packages are named: `content_catalogz_export_YYYY-MM-DD_HH-MM-SS.zip`
